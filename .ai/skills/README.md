Nota: o Laravel Boost pode nomear/posicionar a pasta de skills de forma um pouco diferente
dependendo da versão instalada (ex.: `.ai/skills/`). Após rodar `php artisan boost:install`
no seu projeto real, verifique a pasta que ele criou e mova o conteúdo de `identidade-visual/`,
`crud-padrao/`, `seguranca/` e `testes/` (cada um com seu `SKILL.md`) para o local correto.
Apague este arquivo README.md depois de confirmar o caminho.
